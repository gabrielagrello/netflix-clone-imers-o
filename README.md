🎬 Netflix Clone

Clone da interface da Netflix desenvolvido com HTML5, CSS3 e JavaScript puro, com foco em responsividade, manipulação de DOM e organização de código.

🚀 Sobre o Projeto

Este projeto foi desenvolvido durante uma imersão da Alura com o objetivo de recriar a interface da Netflix, aplicando conceitos fundamentais do desenvolvimento web.

O foco principal foi fortalecer a base em:

Estruturação semântica com HTML5
Estilização moderna com CSS3
Manipulação de DOM com JavaScript
Responsividade
Organização e versionamento com Git
🛠️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript (ES6+)
Git & GitHub
